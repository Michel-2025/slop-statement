The Concept

I Hate Taste Slop is a three-page web statement about AI-generated content — written, visual, and audible. The philosophical thesis you see being typed live on each page is itself generated by an AI. The very thing it criticises.

This is not a bug. It is the point.

The Loop

The thesis does not stop. When it ends, it slides off and begins again — the same words, retyped. AI-generated content does not conclude. It produces, and produces, cycling through the same patterns. The loop is an accurate description of how this works.

You may notice, on your second read, that you are reading the same critique. So is the machine. It has no memory of having said it before.
