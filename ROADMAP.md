## 1\. Project Concept

**SLOP** is a three-page anti-slop web statement built with ironic self-awareness: an AI API generates a live, cursor-typed philosophical thesis criticising AI-generated content — while you're surrounded by examples of exactly that content. The site is simultaneously the critique and the evidence.

> \*"An AI writing about AI writing badly — and doing it in front of you."\*

The site auto-destructs on **August 5, 2026** (one month after the due date) to reduce server energy usage.

## 4\. Feature Breakdown

### 4.1 Live AI Thesis (ThesisEngine class)

* On page load, calls Anthropic API with a system prompt tuned per-page topic
* Streams the response token-by-token using the streaming API
* Types each character with a blinking `|` cursor at \~40ms/char (human reading pace)
* Inline citations appear in smaller text when the AI references a real thinker

  * Format: `— \[Author Name, \*Work Title\*, Year]` injected below the current sentence
* Real references used: Walter Benjamin, Nicolas Carr, Kyle Chayka, Hito Steyerl, Ted Chiang, Lev Manovich
* Each page has a different philosophical angle:

  * Written: authenticity, authorship, the "flatness" of generated prose
  * Visual: Benjamin's aura, reproducibility, algorithmic aesthetics
  * Audible: affect, emotion simulation, the uncanny valley of feeling

### 4.2 Gallery (GalleryRenderer class)

* 3-column Flexbox layout ← **This is the unique UI requirement**
* Responsive: 3-col → 2-col → 1-col at breakpoints
* Each card shows: thumbnail/preview, title, source credit, link
* Click on a card → opens original URL in new tab
* Client-side search/filter bar above gallery
* Loading skeleton states, error states, empty states
* Data source: curated JSON array (15+ real items per page, hardcoded as own content)
* Written page: uses a public API (e.g. NewsAPI or API Ninjas quotes) for supplementary live data

### 4.3 Navigation (NavigationController class)

* Fixed top nav, consistent across all 4 pages
* Logo "SLOP" on the left
* Page links: WRITTEN · VISUAL · AUDIBLE
* Hidden concept link: a tiny `·` or `?` in the corner → links to about.html
* Active page highlighted

### 4.4 Destruction Timer (DestructionTimer class)

* Shows at the very bottom of all pages
* Displays: "This site auto-destructs in X days, X hours, X minutes"
* Target date: August 5, 2026
* Framed as an environmental statement

### 4.5 AI Notice

* Small text at the bottom, above the timer
* "All content on this page has been curated and generated by AI"

## 5\. Design System

### Typography

* **Display/Thesis font:** `DM Serif Display` (Google Fonts) — elegant, literary, accessible serif
* **Body/UI font:** `DM Sans` (Google Fonts) — clean, modern, highly legible, WCAG AA
* Pairing rationale: Both from the DM family → cohesive. Serif for the philosophical "voice", sans for UI chrome.

### Color System (CSS Variables)

```css
:root {
  /\* Page-level backgrounds (overridden per page) \*/
  --bg: #F5F5F0;         /\* Page 1: off-white \*/
  --accent: #1A1A1A;     /\* Text \*/
  
  /\* Shared \*/
  --nav-bg: rgba(255,255,255,0.92);
  --card-bg: rgba(255,255,255,0.7);
  --card-border: rgba(0,0,0,0.08);
  --citation-color: rgba(0,0,0,0.45);
  --cursor-color: #1A1A1A;
  --filter-focus: #333;
  
  /\* Responsive \*/
  --col-gap: 24px;
  --section-padding: 60px;
}
```

Page 2 (blue): `--bg: #D6E8F5; --card-bg: rgba(255,255,255,0.65);`
Page 3 (yellow): `--bg: #FAF0C0; --card-bg: rgba(255,255,255,0.65);`
About (dark): `--bg: #111; --accent: #EEE;`

### Layout

* Max content width: `900px` centered
* Thesis text: `font-size: clamp(1.15rem, 2.5vw, 1.6rem)`, line-height 1.8
* Cursor: `|` blinking with CSS animation
* Gallery: CSS Flexbox, `flex-wrap: wrap`, each card `flex: 0 0 calc(33.333% - var(--col-gap))`

### Accessibility

* All images have `alt` attributes
* Keyboard navigable nav
* Focus rings visible
* Color contrast ≥ 4.5:1 on all text
* `prefers-reduced-motion` respected (cursor still types but no blink)
* Font DM Sans is highly readable, used in major apps

## 6\. API Strategy

### Primary: Anthropic API (claude-sonnet-4-20250514)

* Used for live thesis generation
* System prompt per page, enforces citation format
* Streaming via `fetch()` with `ReadableStream`
* No key exposed (key is Anthropic's system-handled in the artifact context — for deployed version, use Netlify environment variables)

### Secondary: Public Key-Required API

* **Option A: API Ninjas `/quotes`** — fetch philosophical quotes about technology/AI

  * Displayed in the gallery sidebar on the written page
  * Endpoint: `https://api.api-ninjas.com/v1/quotes?category=technology`
* Implement search/filter over the returned quotes
* Handle loading/error/empty states with UI feedback

### Own Content (15+ items per page)

Curated JSON arrays embedded in each page's JS:

* **Written:** 15 real AI-generated or AI-influenced articles (with actual titles, outlets, dates)
* **Visual:** 15 real AI art images (Midjourney, DALL-E, Stable Diffusion examples with source links)
* **Audio:** 15 real AI music/podcast examples (Udio, Suno, NotebookLM, etc.)
